# DP_HW_Hate_speech
Repo for homework at DeepPavlov  
Hate speech classifier.

## Setup and installation
Run 
```console
pip install -r requirements.txt
```
After this you can run 
```
python -m src.main
```
This will upload dataset and prepare it.
Also it will run optimization and train model

After all works done run
```
sh dvc.sh
```
To add all data to dvc repo
